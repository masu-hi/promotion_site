# promotion_site

🎉 **進級制作展 プロモーションサイト**

このリポジトリには、私が制作した「進級制作展」のプロモーションサイトのコードが入っています。
静的 HTML / CSS / JavaScript で構成された Web サイトです。

---

## 🗂 ファイル構成

```
promotion_site/
├─ css/
├─ images/
├─ js/
├─ sass/
├─ index.html
├─ page_contact.html
├─ page_infomation.html
├─ page_photo.html
├─ page_pledge.html
├─ page_price.html
└─ README.md
```

---

## 🚀 使い方

### ローカルで表示する方法

1. このリポジトリをクローン、または ZIP ダウンロードします。

```bash
git clone https://github.com/masu-hi/promotion_site.git
```

2. フォルダ内の `index.html` をブラウザで開いてください。

---

## 📌 ページ一覧

- `index.html`：トップページ
- `page_contact.html`：お問い合わせ
